# Control Flow Based Student Result System

## Description
This Java console application processes student results using control flow statements.

## Features
- Accepts student marks
- Validates marks
- Calculates percentage
- Assigns grades
- Displays pass/fail status

## Concepts Used
- if-else
- switch
- loops
- break & continue
